No installation required! Perfect for beginners and quick edits.

1 Go to Overleaf
Visit www.overleaf.com and create a free account

✅ Pros

  • No installation needed
  • Works on any device

  • Easy to share

⚠️ Cons

  • Requires internet
  • Account needed

💻 Option 2: Local Installation (Full Control)

Install LaTeX on your computer for offline compilation and full customization.

Windows

1 Download MiKTeX
Get it from miktex.org/download
2 Install
Run installer, choose "Install missing packages on-the-fly: Yes"
3 Compile
Open Command Prompt in your CV folder: 
cd path\to\your\cv\folder
pdflatex generated_cv.tex

macOS

1 Download MacTeX
Get it from tug.org/mactex (~4GB)
2 Compile
Open Terminal: 
cd ~/path/to/your/cv/folder
pdflatex generated_cv.tex

Linux

1 Install TeX Live
Ubuntu/Debian: 
sudo apt-get install texlive-latex-base texlive-latex-extra
Fedora: 
sudo dnf install texlive-scheme-basic
2 Compile
pdflatex generated_cv.tex

✅ Pros

  • Works offline
  • Full control
  • Fast compilation
  • No file size limits

⚠️ Cons

  • Large download
  • Setup required
  • Command line usage
🎨 Option 3: VS Code (Best for Developers)

Integrated editing and compilation in Visual Studio Code.

1 Install Requirements
• Visual Studio Code
• LaTeX distribution (MiKTeX/MacTeX/TeX Live)
• LaTeX Workshop extension in VS Code
2 Open & Build
Open your .tex file in VS Code
Press Ctrl+Alt+B (Windows/Linux) or Cmd+Option+B (macOS)
3 View PDF
Press Ctrl+Alt+V or click the preview button

✅ Pros

  • Integrated workflow
  • Syntax highlighting
  • Live preview
  • Error detection

⚠️ Cons

  • Requires local install
  • More setup steps
Download VS Code

🔧 Troubleshooting

⚠️ Common Issues & Solutions
Problem Solution
"Command not found: pdflatex" LaTeX not installed or not in PATH. Reinstall and restart terminal.
"Missing package" errors MiKTeX should auto-install. For TeX Live: sudo tlmgr install <package>
PDF looks incomplete Run pdflatex twice (LaTeX needs two passes)
Special characters broken Check your input data for unescaped special characters
Compilation very slow First time is slow (package installation). Use online tools if persistent.

✨ Customization Tips

Change Paper Size
Replace a4paper with letterpaper in first line
Adjust Margins
Modify: \usepackage[margin=0.75in]{geometry}
Change Font Size
Replace 10pt with 11pt or 12pt
